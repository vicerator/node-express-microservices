Instalacion de node. https://nodejs.org/es/download/

=== GETTING STARTED ===
Sobre la carpeta del package.json
--> npm i 

=== RUN APP ===
--> npm run dev

=== EXTRA COMMAND ===
Comandos: 

(Instalacion de express)
--> npm i express -D

(Instalacion de Nodemon)
--> npm i nodemon -D

(Instalacion de Morgan)
--> npm i morgan -D 

(Lanzar listener Nodemon)
--> npx nodemon index.js
